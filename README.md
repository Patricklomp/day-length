# Day length aoo
#### Code usage order:



#1 Clone repo - 
```
git clone https://github.com/Patricklomp/day-length.git
```
#2 Start backend - 
```
cd day-length/day-length-backend
npm run start
```
#3 Start frontend - 
```
cd day-length/day-length-app
npm run start
```
Open http://localhost:3000 to view it in the browser.
